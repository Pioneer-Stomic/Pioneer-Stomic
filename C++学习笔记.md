### 变量
变量创建语法：数据类型  变量名 = 变量初始值
eg. int a = 10;

### 常量
常量定义方式
1. 宏常量
#define 常量名 常量值
通常在main()前面
2. const修饰的变量
const 数据类型 常量名 = 常量值

## 数据类型
### 整型
数据类型存在的意义：给变量分配和是的内存空间
1. 整型类型：
(1)short(短整型 -32768~32767)
(2)int(整型)
(3)long(长整型)
(4)long long(长长整型)

### sizeof关键字
作用：统计数据类型占用内存大小
语法：sizeof(数据类型/变量)

### 实型（浮点型）
1. 单精度float（7位有效数字）eg. float f1 = 3.14f
2. 双精度double（15～16位有效数字）
注：默认情况下，输出一个小数，无论是float还是double，都会显示6位有效数字
科学计数法：3e2

### 字符型
作用：显示单个字符
语法: char 变量名 = '字母'
注：
1. 显示字符型变量时，用单引号而非双引号
2. 单引号内只能是一个字符，不能是字符串
3. 字符不是直接存入内存，而是存入对应的ASCII码（a-97，A-65）

### 转义字符
作用：表示一些不能显示出来的ASCII字符
\n 换行
\t 水平制表（跳到下一个Tab位置）
\\ 代表一个反斜线字符"\"

### 字符串型
1. C风格字符串 	语法：char 变量名[] = "字符串值"
2. C++风格字符串	 语法：string 变量名 = "字符串值"
注：使用C++风格字符串时要包含头文件 #include <string>

### 布尔类型
bool类型只有两个值：0和1

通过bool{ 表达式 }输出0或1

### 数据的输入
语法：cin >> 变量

### 运算符
作用：执行代码的运算
1. 算术运算符：
(1)四则运算
注：如果a,b为int，则a/b舍去小数部分
(2)取模运算 a % b
(3)递增/递减运算：前置运算（先给变量+1,再进行运算 ++i)； 后置运算（先进行运算，再给变量+1 i++)
2. 赋值运算符：a = 10, num += 6 等价于 num = num + 6
3. 比较运算符：>, =, <, >=, <=
4. 逻辑运算符：!a(非a), a && b(a与b), a || b(a或b)

### 选择结构
1. if语句: if, else if, else (可嵌套)
2. 三目运算符: 根据第一步的真假决定第二步执行哪个语句
语法: 表达式1 ? 表达式2 : 表达式3 (若1为真, 执行2; 若1为假, 执行3)
e.g. 将a与b中较大的值赋给c ==> c = (a > b ? a : b);
注: 三目运算符返回的是变量, 可以继续赋值
3. switch语句: 
语法: 
switch()
switch 1: break;
switch 2: break;
switch 3: break;
default:
注意switch后是冒号
注意: switch判断时只能是整型或字符型, 不可以是区间

### 循环结构
1. while循环语句:

     语法: while(循环条件){ 循环语句 }

      **注: 生成1~100的随机数 rand() % 100 + 1;**
       **在此之前利用系统时间添加随机数种子 srand((unsigned int)time(NULL));** 
       **头文件要加上#include <ctime>**

     注: 可以利用break退出循环

     注: while()中条件为真时才执行, 因此while(1)会一直执行, while(0)永不执行

2. do...while循环语句:

     语法: do{ 循环语句 } while(循环条件);

     注意: 与while的区别在于do...while会先执行一次循环语句, 再判断循环条件

3. for循环语句: 

   语法: for(起始表达式; 条件表达式; 末尾循环体) { 循环语句; }

   注意: for循环中的表达式, 要用分号进行分隔

4. 嵌套循环: 

   外层执行一次, 内层执行一周

### 跳转语句

1. break语句

   作用: 用于跳出选择结构或循环结构

2. continue语句

   作用: 在循环语句中, 跳过本次循环continue之后的语句, 执行下一次循环

   注意: continue不会终止循环, 但break会跳出循环

3. goto语句

   作用: 无条件跳转语句

   语法: goto 标记名;

   ​	  标记名:

   注意: 不建议过多使用, 造成代码逻辑混乱
