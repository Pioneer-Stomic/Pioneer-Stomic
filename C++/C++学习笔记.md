## 数组

定义:存放相同类型数据元素的集合

特点: 1. 数组中每个数据元素都是相同的数据类型; 2. 数组是由连续的内存空间组成的

### 一维数组

三种定义方式: 

1. 数据类型 数组名[数组长度]; 赋值: arr[0] = 1; arr[1] = 2; ...

2. 数据类型 数组名[数组长度] = { 值1, 值2, ...};

3. 数据类型 数组名[] = { 值1, 值2, ...};

注: 第二种方式, 如果{}内不足10个数据, 剩余数据自动初始化为0

注: 第一个数组是arr[0]而不是arr[1]

注: 数组名是常量, 不可以赋值, 例如: arr1 = arr2 会报错

统计数组占用内存大小: sizeof(arr); sizeof(arr[0]);

查看数组首地址: std::cout << arr(显示16进制) ; std::cout << (long long)arr; (转为10进制)

查看数组中某个元素的地址: std::cout << (long long)&arr[0];

### 二维数组

四种定义方式：

```c++
1. int arr[2][3];
2. int arr[2][3] = 
	{
    { 1, 2, 3 },
    { 4, 5, 6 }
	};
3. int arr[2][3] = { 1, 2, 3, 4, 5, 6 };
4. int arr[ ][3] = { 1, 2, 3, 4, 5, 6 };
```

二维数组与内存显示方法同一维数组

## 指针

### 指针的定义

指针是一种用于储存变量内存地址的变量

定义方法：数据类型 * 指针变量名 (int * p)

将内存地址赋给指针：p = &变量名

解引用(*p)：通过内存地址访问指针指向的内存中的数据

*p和num的区别：都指向内存中的数据，但num根据变量的名称访问， *p则通过变量的内存地址访问

指针大小：4字节（x86），8字节（x64）

### 空指针和野指针

**空指针：**指针变量指向内存中编号为0的空间

用途： 初始化指针变量

注意：编号为0的内存拒绝访问，解引用空指针会导致程序崩溃

定义方法：int *p{ nullptr }

**野指针：**指针变量指向非法的内存空间(e.g. int* p = (int *)0x1100)

总结：空指针和野指针都不是可以申请的空间，因此不要访问

### const修饰指针

1. const修饰指针 -- 常量指针(const int* p{ &a }) 指针指向可以改，指针指向的值不可以改
2. const修饰常量 -- 指针常量(int* const p{ &a }) 指针指向不可以改，指针指向的值可以改
3. const既修饰指针，又修饰变量(const int* const p{ &a }) 指针指向和指针指向的值都不可以改

### 指针和数组

int* p{ arr }会得到数组的首地址（第一个数字的内存地址），想要指向下一个数字的内存地址，使用p++



### 指针和函数

值传递只改变形参，不改变实参；地址传递可以同时改变实参和形参 



## 结构体

### 结构体基本概念

结构体属于用户自定义的数据类型，允许用户在结构体内部存储不同的数据类型（属性）

### 结构体定义和使用

语法：

```C++
struct Student
{
  std::string name{ };
  int age{ };
  int score{ };
};
```

通过结构体创建变量的方式：

1. struct 结构体名 变量名

```C++
Student s1;
s1.name = "张三";
s1.age = 18;
s1.score = 100;
```

2. struct 结构体名 变量名 = {属性1, 属性2, ...}

```C++
Student s2{ "李四", 19, 80 };
```

   注：

1. 定义结构体时的关键字时struct，不可省略
2. 创建结构体变量时，关键字struct可省略
3. 通过使用 `.` 运算符（如 `s1.name`）访问结构体内部属性

### 结构体数组

结构体定义：同上

通过结构体数组定义属性：

```C++
Student stuArr[3] = 
{
	{ "张三", 18, 100 },
	{ "李四", 19, 80 },
	{ "王五", 17, 60 }
};
```

通过下表修改属性：

```C++
stuArr[2].score = 65;
```

将第3行的score属性改为65

### 结构体指针

通过指针访问属性

```C++
std::cout << "姓名：" << p->name << '\n';
```

其余同上

### 结构体作函数参数

值传递：直接传入结构体，然后再函数内部通过s.name访问

```
// 定义函数
void printInfo(Student s)
// 调用函数
printInfo(s);
```

地址传递：函数定义一个指针，然后再传入结构体的内存地址，函数内部通过p->name访问

```C++
// 定义函数
void printInfo(Student* p)
// 调用函数
printInfo(&s);
```

### 结构体与const

地址传递时，通过const修饰可以防止实参被修改

```C++
void printInfo(const Student* p)
```

## C++核心编程

### 内存分区模型

1. 代码区：存放函数体的二进制代码，由操作系统管理
2. 全局区： 存放全局变量、静态变量、常量
3. 栈区：由编译器自动分配释放，存放函数的参数值、局部变量等
4. 堆区：有程序员分配和释放。若不释放，程序结束时由从操作系统回收

**程序运行前**
程序编译后生成exe文件，未执行该程序前，有两个区域
1. 代码区：
    ① 存放CPU执行的机器指令
    ② 代码区特点：
    共享：对于频繁被执行的程序，内存中只需有一份代码即可；
    只读：防止程序意外修改其指令
2. 全局区：
    ① 包含：全局变量、静态变量（用static修饰）、全局常量（const修饰的常量）、字符串常量
    ② 不包含：局部变量、局部常量

**程序运行后**

1. 栈区(Stack Area)：

   ①不要返回局部变量的地址，因为函数结束后该地址中的数据被自动释放

2. 堆区(Heap Area)：

   ①通过new运算符在堆区开辟内存(new运算符返回指针变量，需要用指针接收)

   ```C++
   int* p{ new int(10) }; // 单个数据
   int* arr{ new int[10] }; // 大小为10的数组
   ```

   ②需要用delete进行手动释放，否则程序结束后会自动释放

   ```C++
   delete p;
   delete[] arr; // 不要忘记用中括号表示数组
   ```



## 引用

### 引用的基本使用

**作用**：给变量起别名

**语法**：数据类型 &别名 = 原名

```C++
int a{ 10 };
int &b = a; // 可以用a或b操纵同一块内存，a和b的数据类型必须相同
```

### 引用的注意事项

1. 引用必须初始化
2. 引用在初始化后，不可以改变引用对象

### 引用传递

函数传参时，可以利用引用让形参修饰实参

```C++
void swap(int &a, int &b)
{
    int temp{ a };
    a = b;
    b = temp;
}
```
